# Monad ODYSSEY - Navigate the Monadverse

An immersive 3D galaxy visualization of the Monad ecosystem. Explore 100+ projects across DeFi, NFTs, Gaming, Infrastructure, and more - all represented as an interactive cosmic universe.

## Getting Started

### 1. Connect Your Wallet
When you first launch Monad ODYSSEY, you'll be greeted with a login screen. Click **"Connect Wallet"** to authenticate using Privy. This supports:
- Email wallets
- Social logins
- External wallets (MetaMask, WalletConnect, etc.)

### 2. Enter the Galaxy
Once authenticated, you'll be transported into the **Monadverse** - a 3D spiral galaxy where each glowing node represents a project in the Monad ecosystem.

## Navigation Controls

### Camera Controls
- **Click + Drag**: Rotate the camera around the galaxy
- **Scroll Wheel**: Zoom in/out (Min: 5 units, Max: 60 units)
- **Right-Click + Drag**: Pan the camera
- **Auto-Rotate**: The galaxy slowly rotates automatically (stops when you select a project)

### Interacting with Projects
- **Hover**: Hover over any node to see the project name appear above it
  - Node pulses and glows brighter
  - Cursor changes to pointer
- **Click**: Click on any node to open the project details panel

## Visual System

### Color Coding
Each project category has a unique color to help you navigate:

- **DeFi** - Green (`#4ade80`)
- **NFT** - Pink (`#f472b6`)
- **Gaming** - Yellow (`#facc15`)
- **Infrastructure** - Blue (`#60a5fa`)
- **Wallet** - Violet (`#a78bfa`)
- **Social** - Purple (`#836EF9`)

### Galaxy Structure
- Projects are arranged in a **spiral galaxy formation**
- Nearby projects are connected by faint purple lines
- Stars twinkle in the background (5,000 stars)
- Nodes gently pulse and float in space

## Filtering & Search

### Category Filters
Use the filter buttons at the top to view specific project types:
- **All** - Shows all 100+ projects
- **DeFi** - Decentralized Finance protocols
- **NFT** - NFT marketplaces and platforms
- **Gaming** - Blockchain games and play-to-earn
- **Infrastructure** - Core blockchain infrastructure
- **Wallet** - Wallet providers and key management
- **Social** - Social platforms and communication

**Active filters:**
- Highlighted in purple with a glow effect
- Only matching nodes remain visible in the galaxy
- Connections recalculate to show relationships

### Search
- Type in the search bar at the top: **"Search the Monadverse..."**
- Filters projects by name in real-time
- Works in combination with category filters
- Case-insensitive matching

## Project Details Panel

When you click on a project node, a detail panel slides in from the right showing:

### Information Displayed
- **Project Name** - Large title
- **Category Badge** - Color-coded category tag
- **Description** - Brief overview (if available)
- **Status** - Active/Inactive indicator
- **Chain** - Always shows "Monad"

### Actions
- **Launch App** - Opens the project's website in a new tab
- **Close (X)** - Click the X button to close the panel and return to exploring

## Header Bar

**Left Side:**
- **Monad Odyssey Logo** - Rocket icon with title
- **Subtitle** - "ECOSYSTEM DISCOVERY"

**Right Side:**
- **Wallet Address** - Your connected wallet (truncated)
- **Connection Status** - Green pulsing dot indicates active connection
- **Logout Button** - Click to disconnect your wallet

## Footer

**Left Side:**
- **Smart Suggestion** - AI-powered recommendations for trending projects

**Right Side:**
- **Version Info** - Current app version
- **Open Source Link** - GitHub repository access

## Tips for Best Experience

1. **Zoom In** - Get close to see individual project names clearly
2. **Use Filters** - Narrow down by category when looking for specific types of projects
3. **Rotate Freely** - Explore different angles to see the full galaxy structure
4. **Click Nodes** - Each project links directly to their platform
5. **Search First** - If you know what you're looking for, use the search bar

## Technical Details

- **Rendering**: Three.js via React Three Fiber
- **Performance**: Optimized for 100+ simultaneous 3D objects
- **Responsive**: Works on desktop and tablet (mobile support limited)
- **Auto-Save**: Your session persists until you logout
